Carta de Apresentação

Como estudante de Sistemas de Informação...

busco minha primeira oportunidade de estágio para aplicar e expandir meus conhecimentos em desenvolvimento de software e infraestrutura. Tenho sólido domínio de lógica de programação em linguagens como Java (MVC), Python e C++ (OpenGL), além de familiaridade com ambientes Linux (Debian).

Minha trajetória é marcada pela versatilidade: desde o desenvolvimento de algoritmos de álgebra linear até a atuação em projetos de extensão no IFSP, onde lecionei programação para crianças. Essa experiência desenvolveu minha capacidade de comunicação técnica e adaptabilidade, competências que considero essenciais para o trabalho em equipe e a resolução de problemas complexos.

Sou um entusiasta de automação e hardware, sempre priorizando a organização e a documentação clara em meus repositórios no GitHub. Estou pronto para contribuir com dedicação, aprender novas tecnologias e crescer junto à equipe técnica.
